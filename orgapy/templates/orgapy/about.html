Welcome to Orgapy

What is it?

It is a Django application, whose purpose is to act as a notebook. You can keep notes, manage tasks and track objectives. It is open-source, more precisely developped under the MIT License. You can find the source code on GitHub).

Originally, it was developped as an alternative to Google Keep. It now has more advanced features you might want, such as Markdown markup, code syntax highlighting, objective tracking, ...
